build docker image
______________
make docker-build <img-name>
make docker-build traefik-operator-v1


Modify Image Pull Policy for local image usage
______________
imagePullPolicy: IfNotPresent


Add deployment clusterrole & configure rbac kustomization.yaml
______________
config/rabac/deployment_clusterrole.yaml
config/rabac/deployment_clusterrole_binding.yaml


load image into kind cluster
______________
kind load docker-image <img-name> --name <kind-cluster-context>
kind load docker-image traefik-operator-v1 --name edge-cluster


deploy operator
______________
make deploy
